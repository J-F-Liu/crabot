# Crabot

A smart and powerful coding agent.

## Highlights

- No TUI — just a GUI, easy for everyone to use.
- Configure through dialogs — no need to write config files by hand.
- An explicit context window, with every detail customizable.
- Native, high-performance built-in tools.
- Built in pure Rust — single native binary, no runtime dependency, zero GC pauses.
- Each session is saved as a json file in workspace `.agent` folder.
